### HERO.IO App Catalog

HERO.IO is a single-page application (SPA) designed to simulate an app catalog and installation manager. It allows users to browse a collection of productivity and utility apps, search for specific titles, and "install" or "uninstall" them using the browser's LocalStorage for persistence.

The application focuses on demonstrating frontend skills including dynamic DOM manipulation, responsive design, and state management using client-side storage and utilities.

---

### Features

-App Catalog: Browse a list of diverse applications with details like rating, downloads, and size.

-App Details Page: View detailed information about a specific app.

-Installation/Uninstallation: Persist installed apps using LocalStorage.

-My Installation Page: View and manage (uninstall) all currently installed applications.

-Live Search: Filter the app catalog in real-time by app title.

-Loading States: Display a smooth loading animation during page navigation and search operations for a better user experience.

-Toast Notifications: Provide user feedback for successful installations, uninstalls, and errors.

-Responsive Design: Optimized layout for mobile, tablet, and desktop viewing.

### Setup and Usage

To run this project locally, simply clone the repository and open the index.html file in your web browser. Since all data is stored in data.js and persistence uses localStorage, no server environment is required.
